Overview

training quiz API

Training-quiz is a micro-service providing an API to handle quizzes and assessments.

Resources

assessment : A quiz rating

Find by owner

Find an assessment that belongs to an user restricted to TENANT_USER and END_USER_FROM_TENANT users

Request

Example request
GET /api/assessments?owner=ganondorf HTTP/1.1
Authorization: Bearer ganondorf
Host: localhost:8080
cURL request
$ curl 'http://localhost:8080/api/assessments?owner=ganondorf' -i -X GET \
    -H 'Authorization: Bearer ganondorf'
